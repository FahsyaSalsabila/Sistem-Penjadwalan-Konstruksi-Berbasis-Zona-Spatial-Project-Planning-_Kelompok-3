##  **Overview Sistem**

Sistem Penjadwalan Konstruksi Berbasis Zona merupakan aplikasi simulasi yang digunakan untuk mengatur dan memantau proses pembangunan proyek konstruksi yang dibagi ke dalam beberapa zona berbentuk grid 4x4. Setiap zona mewakili jenis pekerjaan seperti pondasi, dinding, atap, atau interior dengan durasi, biaya, dan ketergantungan tertentu. Sistem ini mampu mensimulasikan pengerjaan proyek dari hari ke hari, menentukan urutan pekerjaan berdasarkan ketergantungan antar zona, serta menghitung total waktu dan biaya proyek secara otomatis. Program ini dibangun menggunakan bahasa Java dengan empat kelas utama, yaitu *Zona*, *PetaProyek*, *Proyek*, dan *MainKonstruksi* yang saling berinteraksi untuk menampilkan progres pembangunan secara efisien dan terukur.
